Cehavl

Cehavl est un plugin Neovim pour mélanger l’ordre des lettres dans un mot. Par exemple, "cheval" peut devenir "cehavl", ou "haevcl".
Work in progress. Répo git

Power Nuts

Power Nuts est un projet de plugin Neovim pour l’utilisation de l’assistant de preuve Squirrel. Work in progress.
